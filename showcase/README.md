# Milo Showcases

Showcases are complete demonstrations of Milo's product thesis. They are
designed to exercise several framework surfaces together and to provide
recording-quality proof, rather than to be copied into a new project.

For small, focused copy paths, use [`examples/`](../examples/) instead.
